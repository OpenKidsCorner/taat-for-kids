### Decision — Creative Writing App

After discussion and a vote, the group decided to build an app that helps with creative writing — the kids were taking creative writing classes at the time, which made it immediately relevant.

The next step was to identify the features before writing the prompt. The prompt the family came up with together:

```
Create a web application that helps us improve creative writing through the following features:
- Suggest topics for writing stories
- Provide stories for reading and ask students to improve the writing
- Give hints for next steps (e.g. starting, body and conclusion)
- Track and suggest new vocabulary words for different grades
- Student can submit and teacher can review
Create sample data too so that we can explore and test the app
```

## Key Observations

- The initial reluctance to suggest ideas was familiar — the same hesitation seen in the classroom. A small push was enough to unlock real, thoughtful suggestions. The ideas that came out once the conversation opened up were genuinely interesting and ranged from practical tools to meaningful learning aids.

- Building the prompt as a group — listing features before writing it — was itself a valuable exercise. Deciding what to ask for requires understanding what you want, and that is not always obvious until you try to say it clearly.

- Running the same prompt across three different tools produced three different results. That comparison alone is worth exploring — the same natural language, interpreted differently by different systems, producing different outputs.
