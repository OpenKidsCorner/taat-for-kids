# Technology as a Tool (TaaT) for Kids

بسم الله الرحمن الرحيم

If Allah wills, this repository contains notes from our weekly sessions where kids learn about Technology as a Tool (TaaT) to serve Allah. May Allah accept this effort, forgive our shortcomings, and grant us sincerity.

## Sessions

| # | Date | Topic | Learning Outcomes | Notes |
|---|------|-------|-------------------|-------|
| 1 | April 04, 2026 | Introduction to Technology — Language, Devices & Robot Navigation (~4.25 hrs) | Every device speaks a fixed, limited language; extending what a machine can do means defining new instructions precisely | [Session 1](sessions/session-01/README.md) |
| 2 | April 11, 2026 | The Tower of Computer Languages — Binary, ASCII & Encoding (~4.5 hrs) | Computers speak only binary at the hardware level; numbers map to letters via ASCII; meaning is built layer by layer up a tower of languages | [Session 2](sessions/session-02/README.md) |
| 3 | April 19, 2026 | Path Descriptions, Binary Arithmetic & Assembly Language | The same path can have multiple valid solutions; binary carrying must be understood not memorized; Assembly adds a human-readable layer above binary using a small fixed vocabulary | [Session 3](sessions/session-03/README.md) |
| 4 | April 25, 2026 | The Full Language Tower — From Binary to Natural Language & First Hands-on with Code | The tower spans from binary to natural language; a single AI prompt can produce hundreds of lines of working code; every layer enforces its own rules and even one small deviation produces an error | [Session 4](sessions/session-04/README.md) |
